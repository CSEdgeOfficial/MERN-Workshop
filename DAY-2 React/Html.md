
---

### **Foundational & Text Tags**

| Tag | Name | Usage |
| --- | --- | --- |
| `<html>` | Root | The main container for the entire webpage. |
| `<head>` | Metadata | Contains info about the page (title, styles, scripts). |
| `<body>` | Body | Contains all visible content like text and images. |
| `<h1>` - `<h6>` | Headings | Creates titles; `<h1>` is the largest, `<h6>` the smallest. |
| `<p>` | Paragraph | Defines a block of plain text. |
| `<br>` | Line Break | Forces a new line without starting a new paragraph. |
| `<hr>` | Horizontal Rule | Inserts a horizontal line to separate content. |

---

### **Formatting & Multimedia Tags**

| Tag | Name | Usage |
| --- | --- | --- |
| `<strong>` | Bold | Makes text **bold** to show importance. |
| `<em>` | Emphasis | Makes text *italicized* for emphasis. |
| `<a>` | Anchor | Creates a link to another page (uses `href`). |
| `<img>` | Image | Displays an image (uses `src` and `alt`). |
| `<ul>` | Unordered List | Creates a bulleted list. |
| `<ol>` | Ordered List | Creates a numbered list. |
| `<li>` | List Item | The individual item inside a list. |

---

### **Structural & Input Tags**

| Tag | Name | Usage |
| --- | --- | --- |
| `<div>` | Division | A generic container used to group sections of a page. |
| `<span>` | Span | A container used to style small parts of text inline. |
| `<button>` | Button | Creates a clickable button. |
| `<form>` | Form | A container used to collect user input. |
| `<input>` | Input | A field where users can type data or select options. |
